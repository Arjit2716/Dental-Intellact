# GLB-Dental-Intellect

This is a Next.js starter project for GLB-Dental-Intellect, bootstrapped with Firebase Studio.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Deployment

To deploy your Next.js app, you can use the [Firebase CLI](https://firebase.google.com/docs/cli).

First, build the application:
```bash
npm run build
```

Then, deploy the application:
```bash
firebase deploy --only hosting
```
